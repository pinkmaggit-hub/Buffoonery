# Buffoonery
Um mod de balatro que adiciona conteúdo adjacente ao original enquanto mantém um estilo próprio.

- 34 Curingas
  > 9 Comuns<br> 10 Incomuns<br> 5 Raros<br> 9 Speciais*<br> 1 Lendário
- 4 Baralhos
  > Com suas 4 respectivas [Capas](https://github.com/larswijn/CardSleeves)
- 2 Aprimoramentos
  > Aplicados por 1 carta de Tarô.
- 2 Apostas
  > Depois da Aposta Dourada

<br>**<ins>Recomendado</ins>: Baixe a [última versão](https://github.com/pinkmaggit-hub/Buffoonery/releases)!**

### Sleeves
![Buffoonery Sleeves](repo/Sleeves.png)

[Card Sleeves](https://github.com/larswijn/CardSleeves) apply their respective Baralhos effect.
Combined with their Baralho, they apply a different Efeito:

| Nome | Baralho + Sleeve |
| ---- | ---- |
| Capa JimboStation | Start run with<br>**Deluxe Memory Card**<br>Curinga instead |
| Capa Galopeira | Start run with<br>**White Pony**<br>Curinga instead |
| Capa de Pordcelana | Start run with<br>**Dorkshire Gold**<br>Curinga instead |
| Capa de Arenito | 37.5% lowered Win Ante<br>X3 base Blind size |
